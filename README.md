# Assignment Description
This assignment focuses on the implementation and application of Stack and Queue data structures using the C programming language.
The objective is to understand how stacks and queues work and how they can be used to solve real-world problems.
The programs included in this assignment are:
1. Reverse a string using Stack  
2. Check for Balanced Parentheses using Stack  
3. Find Next Greater Element using Stack  
4. Simulate a Printer Queue using Queue operations  
5. Implement Circular Queue with basic operations (Enqueue, Dequeue, Peek, Display)
Each program demonstrates the use of fundamental data structure concepts such as insertion, deletion, traversal, and problem-solving using stacks and queues.
